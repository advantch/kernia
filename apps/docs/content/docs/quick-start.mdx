title: Quick Start

## 1. Scaffold

```bash
pip install kernia kernia-cli kernia-fastapi kernia-sqlalchemy
kernia init --adapter sqlite --framework fastapi
```

This creates an `auth.py` with a minimal `init(...)` call and an `.env.example`
with a generated secret.

## 2. Generate and run migrations

```bash
kernia generate
kernia migrate
```

`generate` merges the core schema and every plugin schema into one Alembic
revision. `migrate` applies it to the configured database.

## 3. Mount Kernia

```python
from fastapi import FastAPI
from kernia_fastapi import mount_auth
from auth import auth

app = FastAPI()
mount_auth(app, auth, prefix="/api/auth")
```

Run the app:

```bash
uvicorn main:app --reload
```

## 4. Use the routes

The default email/password setup exposes:

```http
POST /api/auth/sign-up/email
POST /api/auth/sign-in/email
POST /api/auth/sign-out
GET  /api/auth/get-session
POST /api/auth/forget-password
POST /api/auth/reset-password
```

Session cookies are HttpOnly and use Better Auth-compatible wire defaults.

## 5. Add plugins

```python
from kernia.plugins.email_password import email_and_password
from kernia.plugins.organization import organization
from kernia.plugins.open_api import open_api
from kernia_api_key import api_key

plugins = [
    email_and_password(),
    organization(),
    api_key(),
    open_api(),
]
```

Each plugin contributes routes, hooks, schema, or secondary behavior.
