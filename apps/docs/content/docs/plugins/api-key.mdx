---
title: API Key
description: Create, list, verify, and revoke API keys.
icon: KeyRound
---

```bash
pip install kernia-api-key
```

```python
from kernia_api_key import api_key

plugins=[api_key()]
```

Routes:

```http
POST /api-key/create
GET  /api-key/list
POST /api-key/revoke
POST /api-key/verify
```

API keys are hashed before storage and are only shown in full at creation time.
