Kernia is an independent Python implementation compatible with the Better Auth
wire protocol. It keeps the browser-facing contract that Better Auth clients
depend on, while exposing Python packages, imports, adapters, and deployment
patterns.

The project is designed for Python backends that need production-grade auth
without losing access to the Better Auth ecosystem.

## What compatibility means

Kernia preserves route shapes, cookie names, response payloads, and client-visible
defaults where Better Auth clients depend on them. That is why a JavaScript app
can keep using `better-auth/client` while the server runs Python.

Compatibility does not mean this repository vendors Better Auth source. Kernia
uses its own Python code, package names, docs, examples, tests, and release
surface.

## Package names

The public Python package family uses `kernia` names:

```bash
pip install kernia kernia-fastapi kernia-sqlalchemy kernia-stripe
```

```python
from kernia import KerniaOptions
from kernia.auth import init
from kernia.plugins.email_password import email_and_password
```

## What ships today

- Core auth routes for email/password, sessions, password reset, accounts, and
  cookies.
- Built-in plugins for organizations, admin, magic links, email OTP, passkeys,
  SSO, SCIM, API keys, OpenAPI, two-factor auth, SIWE, anonymous auth, one-tap,
  username, phone number, and more.
- Adapters for memory, SQLAlchemy, Mongo, and Redis-backed secondary storage.
- Integrations for FastAPI, Starlette, and Django.
- A CLI for app scaffolding, migrations, secrets, diagnostics, and schema
  generation.
- A full SaaS demo that exercises login, logout, settings, API keys, admin
  config, Stripe sync, billing checks, and usage surfaces.

## When to use Kernia

Use Kernia when your product backend is Python, but your frontend or auth
surface benefits from Better Auth-compatible client behavior. It works best for
teams that want to own their auth server, database, migrations, and plugin
surface instead of sending auth state to a hosted provider.
