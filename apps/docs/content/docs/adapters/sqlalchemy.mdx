---

```bash
pip install kernia-sqlalchemy
```

```python
from kernia_sqlalchemy import sqlalchemy_adapter

database = sqlalchemy_adapter("postgresql+asyncpg://...")
```

The SQLAlchemy adapter is the recommended relational adapter for production.
It supports the migration workflow generated by `kernia generate`.
