# Kernia Docs

Kernia's standalone Fumadocs + Next.js documentation site.

The app uses the same Fumadocs-based docs shell, theme-token approach,
landing-page structure, custom sidebar, MDX rendering, and Geist font setup as
the Better Auth documentation, adapted for Kernia's Python packages and public
API names.

```bash
pnpm install
pnpm dev
pnpm build
pnpm typecheck
```

The Vercel project root directory should be `apps/docs`.
